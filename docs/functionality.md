Functionalities:

1. register - username, email, password (hashed)
2. login & logout - flask session management
3. edit Profile page - avatar, bio, total check-ins, and districts visited
4. view others' profile
5. create posts (pins) - can pins on maps and post an article with title, description, rating and maybe photos
6. can delete any own post (pin)
7. Homepage map - leafles.js, photos, pins, maps, comments
8. check-in (post) detial pages - full descritpion, photo, comments of a post
9. pick on Map button — mini Leaflet map inside the check-in form
10. Expore page - browse all check-ins (posts), filtered by category or rating, and sort by date or rating
11. search page - keyword search across check-in titles or description
12. save & favourite - personal saved list
13. Comments - post a comment in a check-in
14. leaderboard - rank users by check-ins, districts explored, avg rating
