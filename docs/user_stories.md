# User Stories

> This file contains the user stories for the project.
> Format: As a [role], I want to [action], so that [benefit].
> 10 user stories

---
1. As a visitor, I want to see an interactive map on the homepage that displays all check-in locations as colour-coded markers, so that I can quickly get a visual overview of where people have been around Perth and UWA.
2. As a logged-in user, I want to fill in a form with a place name, photos, title, and description, so that I can share a location I have visited with the community.
3. As a user creating a check-in, I want to click a "Pick on Map" button to open a small interactive map and drop a pin on the exact spot, so that my check-in is pinned to the correct coordinates rather than just a text address.
4. As a visitor browsing the homepage map, I want each check-in marker to use a distinct colour based on its category, so that I can instantly tell what type of place a pin represents without clicking it.
5. As a new user, I want to register an account with a username, email, and password, so that my check-ins are saved under my own profile and others can find my posts.
6. As a logged-in user, I want to view my profile page showing my avatar, bio, total check-ins, and districts visited, so that I can track my own exploration history in one place.
7. As a logged-in user, I want to save a check-in posted by another user, so that I can keep a personal list of places I want to visit later.
8. As a logged-in user, I want to leave a comment on a check-in, so that I can share my own experience of that place or ask the author a question.
